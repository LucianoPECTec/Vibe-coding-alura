# Vibe-coding-alura
Meu primero vibecoding para acompanhamento dos dados das escolas de São José do Rio Preto

Aqui o conceito é otimizar as informaçõs coletadas pelo B.I ( Escola total)
Subindo os arquivos xml obtidos semanalmente das plataformas, o programa já faz o filtro de prioridades para aquela semana, tanto em escolas, turmas e alunos.
O ponto chave é criar um gráfico de evolução das escolas de acordo com as formações e acompanhamentos realizados pelos PECs
